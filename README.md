# Fork & Pull Request Test

## Overview
This repository is used to verify basic GitHub collaboration skills, including working with forks, branches, commits, and pull requests.

## Task
Contributors are expected to make a single, small change (e.g. add their name to `CONTRIBUTORS.md`) and submit it via a pull request.

## Guidelines
- One pull request per contributor  
- Use a feature branch    

## Completion
A merged pull request confirms successful completion of the task.
